编译安装PHP

发表于 2016-05-31   |  
  • 先安装依赖

  • 新建系统用户 useradd www -s /sbin/nologin

  • 安装PHP

    1. 下载PHP，解压进入

    2. 编译参数

    3. 出现的一些错误

      • configure: error: jpeglib.h not found

        去网站 http://www.ijg.org/ 下载源码。我用的版本是jpegsrc.v8d.tar.gz，下载解压进入。
        ./configure --prefix=/usr/include/jpeg; make && make install

      • configure: error: mcrypt.h not found. Please reinstall libmcrypt.

        去网站http://mcrypt.hellug.gr/lib/index.html 的下载源码，我下的版本是libmcrypt-2.5.7.tar.gz; 下载解压进入。
        ./configure; make && make install
        重新编译php，这时增加参数--with-mcrypt

      • [参考来源]：(http://www.cnblogs.com/alexqdh/archive/2012/11/20/2776017.html)

    • 编译完成后

      1. 复制 php.ini 到安装目录

      2. 添加php-fpm到系统启动项

    • 其他配置

    决定是否暴露 PHP 被安装在服务器上（如: 在 Web 服务器的信息头中加上其签名：X-Powered-By: PHP/5.3.7)

    • 开启opcache
